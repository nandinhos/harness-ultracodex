# 06 - Hooks e Automacoes

Hooks servem para bloquear ou alertar antes de erros recorrentes.

| Hook | Objetivo |
|---|---|
| `pre-action-risk-check.sh` | Classificar risco antes de agir. |
| `pre-edit-scope-check.sh` | Evitar edicoes fora do escopo. |
| `pre-finish-evidence-check.sh` | Impedir conclusao sem evidencia declarada. |
| `destructive-command-guard.sh` | Bloquear comandos destrutivos sem confirmacao. |

## Principio

Automatize o que for mecanico. Use skill quando o problema exigir julgamento.
